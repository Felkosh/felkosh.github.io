PowerGuide

PowerGuide is a solution for creating, editing and using interactive 3D manuals in digital format. Interactive binding of 3D model and technical documentation plus simultaneous process animation make assembly, maintenance and repair operations as simple as possible and offer visual teaching methods for personne

Usage

Teams can quickly create and update high-quality graphics and develop the following types of technical documentation:

User, maintenance and repair manuals

Data continuity helps to faster proceed to problem solving and decision making

Standard operating procedures for assembly and installation operations

Interactive binding of 3D-model, reference information and step-by-step process animation speed up the assembling of complex products

Brand new training systems

Maximum visualization and engagement in the learning process

E-catalogs — interactive specifications and parts lists

Reference information showing the technical peculiarities of using the product and a complete set of documentation in interactive digital format

Functional strengths

Create and publish manuals

Import source 3D models. If the design has changed, just upload a new version of the model and the animation will be updated automatically

Built-in web editor

The tool allows you to handle the structure of manuals, 3D animations, drawings and texts

Integration with a VR helmet

Full immersion into the assembling and repairing process

Interactive 3D model-to-reference information binding

The 3D model automatically uploads drawing, specification and DD with all information about the product or process being available on one screen

Control of versions and revisions

You get a single source of up-to-date data available throughout the company. Make sure that data are up-to-date and take advantage of a revision history feature
