VEMINA

Custom aircraft interiors

Solution for automating the design of interior parts and components of aircraft with automatic generation of the furniture structure of a 3D model.

Customer

VEMINA Aviaprestige

Challenge:Large number of routine operations.
Legacy methodology made it very difficult to standardize and re-use technical solutions and solutions that have already been applied in finished projects. Standard functional components needed to be designed every time from the ground.

Solution description

Creation of new operations:

  • Adaptive template for honeycomb panel
  • Adaptive templates for furniture standard components
  • Furniture hardware and components inserted in model as an assembly integrated to adjacent materials
  • Tools for automatic fasteners arrangement
  • Creation of coordinate systems relative to selected points and location of the necessary components at these points (bushings, pegs, etc.)

Result

Time for drafting the design documentation has been reduced from 2 days to 4 hours. As a result employees have received more time for innovative development.

RUBIN

Central Design Bureau
for Marine Engineering

Solution for drafting and adaptation of hull structure drawings in CATIA

Customer

Central design Bureau for marine engineering RUBIN

Challenge: The complexity of the process of designing hull structures in Catia v5. The lack of the necessary functionality for the design and execution of shipbuilding drawings.
More than 25 view and 1000 positions are in the project.
Preparation of 1 view takes up to 5 days.

Solution description

Solution description

Solution for drafting and adaptation of hull structure drawings in CATIA

Creation of new operations:

  • Automatic adaptation of views and sectional views of hull structures drawings
  • Automatic creation of positions
  • Automatic leader line creation
  • Checking relevance of drawing views
  • Base plane display

Solution description

Solution for drafting drawing views for piping, ventilation and air conditioning systems in CATIA

Creation of new operations:

  • Creating position callouts on a pipeline assembly drawing
  • Creating callouts for fixture element
  • Cleaning invisible center tubes
  • Automatic dimensioning in the pipeline assembly drawing

Result

Reducing the time of release of drawings
and design documentation several times.
View creation time decreased
to 5-10 minutes.
